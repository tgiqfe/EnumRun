# パターン1

ほぼ最小限の構成。

  • ファイル、ログ、標準出力の各ファイル保存先は%TEMP%\EnumRun配下

  • 連続起動禁止期間⇒60秒
  • ログの保持期間⇒10日
  • 最小ログレベル⇒Info
