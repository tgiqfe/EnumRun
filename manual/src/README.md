# EnumRun

キックスタート用ツール

<img src="./top01.drawio.svg" width="500">

開始用の``.exe``ファイルを実行することで、事前設定したフォルダー配下のスクリプトファイルを実行します。

## システム要件

### OS

以下のOSで動作確認をしています。
- Windows 10 (Ver. 1809以上)
- Windows 11
- Windows Server 2019
- Windows Server 2022
